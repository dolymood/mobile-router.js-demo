## How can i see it?

1) Get it:

	git clone https://github.com/dolymood/mobile-router.js-demo.git

2) Run it:
	
	cd mobile-router.js-demo
	npm install
	npm start

Now you can see it `http://127.0.0.1:3000` and `http://127.0.0.1:3000/v2/`.
